##### pass
- `pass` 
	- 用于空语句, 它用于表示一个代码块中暂时不做任何操作, 或者在语法上需要一个语句, 但实际上不需要执行任何操作
```python
def placeholder_function():
    pass  # 这里暂时不做任何操作，使用 pass 作为占位符

if condition:
    # 如果条件满足，执行一些代码
    # 但是现在暂时没有具体实现
    pass

for i in range(5):
    # 在循环中暂时不执行任何操作
    # 使用 pass 作为占位符
    pass

class EmptyClass:
    # 这是一个空类，暂时没有任何方法或属性
    # 使用 pass 作为占位符
    pass

```
