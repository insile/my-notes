##### 表达式语句
- 表达式语句
	- 表达式语句是由[[C.表达式|表达式]]组成并以分号 `;` 结尾的语句. 它们是 C 语言中最常见的一种语句类型, 用于执行计算, 调用函数和修改变量值等操作. 在执行表达式语句时, 将根据表达式概述的规则来计算, 并在执行下一个语句前, 完成表达式计算的所有副作用
- 语法
```c
表达式;
// 赋值表达式: 用于将值赋给变量
// 函数调用表达式: 用于调用函数
// 增量和减量表达式: 用于递增或递减变量的值
// 算术表达式: 用于执行算术运算
// 逻辑表达式: 用于执行逻辑运算
// 复合表达式: 组合多种表达式
```
- 示例
```c
x = ( y + 3 );
x++;
x = y = 0;
proc( arg1, arg2 );
x = y = ( f( x ) + 3 );
```
